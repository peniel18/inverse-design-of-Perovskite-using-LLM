# Docs

This directory holds the reference documentation for the current generic relay system.

- `architecture.md`
  Process model, main scripts, and control flow.
- `config-reference.md`
  Variable-by-variable configuration reference.
- `operations.md`
  Startup, restart, commands, mirroring, notifications, exit codes, and troubleshooting.

Start here if you need the full behavior of the codebase, not just the setup path.
